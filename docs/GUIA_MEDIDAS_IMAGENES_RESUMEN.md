# Guía rápida de tamaños de imagen

## Logo del sitio
- Tamaño recomendado: `800x800 px`
- Ratio: `1:1`
- Formato: `PNG` o `WebP`
- Usar espacio interior y fondo transparente si es posible.

## Favicon
- Tamaños: `32x32`, `48x48`, `512x512`
- Ratio: `1:1`

## Slider / hero
- Tamaño recomendado: `1920x1080 px` o `2000x1000 px`
- Ratio: `16:9` o `2:1`
- Guardar el contenido importante centrado.

## Noticias / tarjetas
- Tamaño recomendado: `1200x675 px`
- Ratio: `16:9`
- Se recorta con `object-cover`, mantener el foco central.

## Post destacado e individual
- Tamaño recomendado: `1600x900 px` o `1920x1080 px`
- Ratio: `16:9`
- Buena calidad y anchura amplia para pantallas grandes.

## Open Graph / social
- Tamaño recomendado: `1200x630 px`
- Ratio: `1.91:1`

## Nota
- El proyecto usa `Spatie Media Library` y `Tailwind CSS` con `object-cover`.
- Favor entregar imágenes horizontales con el contenido principal bien centrado.
